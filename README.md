# SpellingQuiz

## Purpose

The `spellingQuiz.py` script helps automate the generation of Spelling Quizzes by pulling from a bank of spelling words categorized by grade level found in the Spelling_Lists folder. Every time the script is ran, the chosen words are then removed from the related grade level spelling list file while the full list of words are stored in the Master_Lists folder.

### === WIP ===

The `spellingTest.py` script helps automate the generation of a Spelling Test by pulling either the first 5 words or the last 5 words from the last 4 quizzes based on whether the number of the test is even or odd (i.e. Spelling Test #1 pulls the first 5 words while Spelling Test #2 pulls the last 5 words).
